# Task Template for Development Agents

Use this template for every implementation task.

## Task

Implement: `<short task name>`

## Required reading

- `AGENTS.md`
- `docs/CODEX_WORKFLOW.md`
- `docs/PR_REVIEW_CHECKLIST.md`

## Allowed files

List the exact files or directories allowed for this task. If another file must be changed, explain why in the PR body.

## Product rules

List the business rules that must be true in the finished product.

## Acceptance checks

List the exact manual and automated checks required before opening the PR.

## Out of scope

List features that must not be implemented in this task.

## Required commands

Run:

```bash
npm run build
npm test
```

## PR requirements

The PR must include:

- Changed files
- Reason for each changed file
- Product rules implemented
- Build result
- Test result
- Out-of-scope items
